- 53: DNS
- 67: DHCP Server
- 68: DHCP Client
- 69: TFTP
- 123: NTP
- 161: SNMP Agent
- 162: SNMP Manager
- 514: Syslog
- 1985: HSRP
- 3222: GLBP
- 5246: Control tunnel
- 5247: Data tunnel

### TCP

- 20: FTP data
- 21: FTP control
- 22: SSH
- 23: Telnet
- 25: SMTP
- 53: DNS
- 80: HTTP
- 110: POP3
- 443: HTTPS
- 8140: Puppet
